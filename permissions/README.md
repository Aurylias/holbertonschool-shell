All my work for the Shell permissions :

0-iam_betty -> Change the current user to "betty".

1-who_am_i -> Show in the terminal the current user.

2-groups -> Show all the groups the current user is part of.

3-new_owner -> Change the owner of the hello file to "betty".

4-empty -> Create a empty file named "hello".

5-execute -> Give the owner the permission to execute the "hello" file.

6-multiple_permissions -> Change the permissions of the "hello" file so the owner have all of them, the group can read and execute it, and other users can read it.

7-everybody -> Give to all user the permission to execute the "hello" file.

8-James_Bond -> Remove all permissions to the owner and the group but give all of them to other users for the "hello" file.

9-John_Doe -> Give the owner all permissions, the permissions to read and execute for the group, and write and execute permissions for all other users, for the "hello" file.

10-mirror_permission -> Set the permissions of "olleh" the same of the "hello" file.

11-directories_permissions -> Give to all users the permission to execute, from the current directory to all subdirectories.

12-directory_permissions -> Create a directory named "my_dir" with all permissions for the owner, read and execute permissions for the group and only execution for all other users.

13-change_group -> Change the group of the "hello" file to "school".

14-change_owner_and_group -> Change the owner to "vincent" and the group to "staff" for all files and directories from the current working directoy.

15-symbolic_link_permissions -> Change the owner to "vincent" and the group to "staff" for the "_hello" file, which is a symbolic link.

16-if_only -> Change to owner of the "hello" file to "vincent" only if the current owner of the "hello" file is "guillaume".
