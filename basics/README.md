All my work for the Shell basics :

0-current_working_directory -> Show the current working directory.

1-listit -> List all files and directories in the current working directory.

2-bring_me_home -> Return in the home directory.

3-listfiles -> List in long format all files and directories in the current working directory.

4-listmorefiles -> List in long format all files and directories (even hidden ones) in the current working directory.

5-listfilesdigitonly -> List in long format and with digit, all files and directories (even hidden ones) in the current working directory.

6-firstdirectory -> Create a directory in /tmp/my_first_directory.

7-movethatfile -> Move the "betty" file in /tmp to the "my_first_directory" directory, also in /tmp.

8-firstdelete -> Remove the "betty" file from the "my_first_directory" in the /tmp.

9-firstdirdeletion -> Remove the "my_first_directory" and all it's files and subdirectories from /tmp.

10-back -> Change the current working directory to it's parent.

11-lists -> List in long format all files and directories (even hidden ones) in the current working directory, it's parent and even those in the /boot.

12-file_type -> Print in the terminal the type of "iamafile".
